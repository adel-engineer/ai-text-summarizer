# AI Text Summarizer

A simple full-stack web application that summarizes long text using Artificial Intelligence.

## Features
- Paste any long text
- Click "Summarize"
- Get a short and clear AI-generated summary
- Clean and simple interface

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- Node.js
- Express.js
- OpenAI API

## How It Works
1. The user enters text in the web page.
2. The text is sent to the server.
3. The server sends the text to the OpenAI API.
4. OpenAI returns a summary.
5. The summary is displayed on the page.

## Installation

1. Clone the repository:
git clone https://github.com/your-username/ai-text-summarizer.git

2. Go into the project folder:
cd ai-text-summarizer


3. Install dependencies:
npm install


4. Create a `.env` file and add your API key:
OPENAI_API_KEY=your_api_key_here


5. Run the server:
node server.js


6. Open in your browser:
http://localhost:3000
